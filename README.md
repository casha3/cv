## Проекты

### Fuzzy Hash Signature Search
Инструмент для поиска сигнатур на основе fuzzy hashing (ssdeep, tlsh). Реализовано индексирование и быстрый поиск похожих файлов в больших наборах данных. Применяется для обнаружения модифицированных вредоносных образцов.

**Стек:** `Python` `fuzzy hashing` `Elasticsearch`

### Automated Workspace
Автоматизированная среда разработки с интеграцией статического и динамического анализа. Pipeline включает SAST (Semgrep, Bandit), SCA (Trivy), secrets scanning (Gitleaks) и fuzzing (AFL++) в рамках CI/CD (GitLab CI).

**Стек:** `Python` `Docker` `GitLab CI` `Semgrep` `Trivy` `Gitleaks` `AFL++`
